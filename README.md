# Elite Service Reliability Command

A white, black, and `#f4af00` gold React incident-operations workspace for **Elite Era Development L.L.C.** It tracks incidents, tiered SLA deadlines, breach risk, client impact, assignment, recovery work, root cause, and prevention.

## Workflow

```text
Client issue
→ Incident intake
→ SLA calculation
→ Skill-based assignment
→ Recovery checklist
→ Client update
→ Root-cause review
→ Prevention action
```

## Features

- Tiered Platinum, Gold, and Standard SLA monitoring
- Safe, Watch, At risk, Breached, and Met incident signals
- Smart assignment based on skills, capacity, and availability
- Incident workbench with recovery checklist and client updates
- Client-safe status page
- Root-cause and prevention tracking
- MTTA, MTTR, service-health, and repeat-issue reporting
- TXT, JSON, and print/PDF export
- Local browser persistence
- Vitest specifications and GitHub Actions workflow

## Run locally

```bash
npm install
npm run dev
```

